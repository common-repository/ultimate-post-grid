Ultimate Post Grid is a WordPress plugin. Ultimate Post Grid helps to display post and custom post type with grid system anywhere in the website. Responsive and has strong backend controlly system so user can make changes easily.

== Description ==

Fully Responsive.
3 Different Layout.
Support Multiple Column.
Ability to set order by options.
Display Any Post Types.
Pagination support.
Pagination Text Option.
Pagination Text Color Option.
Pagination Background Color Option.
Custom Image Size Option.
Show/Hide Title Option.
Title Color Option.
Title Font Size Option.
Content Color Option.
Show/Hide Date Option.
Date Color Option.
Excerpt Length Option.
Excerpt Length Color Option.
Image Opacity Option.
Show/Hide Content Option .

= HOW IT WORKS? =
* Display post as grid 
* Display custom post type as grid

== Key Features ==

* User friendly backend control system
* Highly customizable
